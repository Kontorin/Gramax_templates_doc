# Руководство пользователя  
Автоматизированная система расчёта заработной платы и премий сотрудников  
(АСРЗП)

Документ разработан в соответствии с ГОСТ 19.505–79, ГОСТ 19.506–79 и ISO/IEC 26514.

---

# 1. Назначение документа

Настоящее Руководство пользователя предназначено для сотрудников финансового отдела, руководителей подразделений и других пользователей, которые работают с автоматизированной системой расчёта заработной платы и премий сотрудников (АСРЗП).

Документ содержит сведения о назначении системы, её возможностях, порядке установки, начальной настройке и правилах эксплуатации.

---

# 2. Область применения

АСРЗП используется для автоматизации следующих процессов:

- ведение данных сотрудников;
- расчёт окладной части заработной платы;
- расчёт налогов и удержаний;
- начисление премий различного типа;
- формирование итоговых расчётных ведомостей;
- экспорт данных в различные форматы (PDF/XLSX);
- просмотр истории изменений.

Система применяется на рабочих местах финансовых специалистов в пределах внутренней корпоративной сети.

---

# 3. Общие сведения о системе

АСРЗП представляет собой клиент–серверное приложение, состоящее из:

- клиентской части (пользовательский интерфейс);
- серверной части (модули расчёта и обработки данных);
- базы данных PostgreSQL.

Интерфейс разработан для пользователя без специальной технической подготовки.

---

# 4. Требования к рабочему месту пользователя

## 4.1. Технические требования

- Операционная система: Windows 10/11.  
- Оперативная память: ≥ 8 GB.  
- Процессор: ≥ 2.5 GHz.  
- Свободное место на диске: ≥ 1 GB.  
- Доступ к корпоративной сети.

## 4.2. Программные требования

- Установленный корпоративный клиент АСРЗП.
- Доступ к внутреннему домену.
- Корпоративные учётные данные.

---

# 5. Установка и запуск системы

## 5.1. Установка

1. Получите установочный пакет у администратора или скачайте его из внутреннего репозитория.  
2. Запустите файл `setup.exe`.  
3. Следуйте инструкциям мастера установки.  
4. После завершения установки убедитесь, что ярлык “АСРЗП” появился на рабочем столе или в меню Пуск.

## 5.2. Первичный запуск

1. Запустите приложение с помощью ярлыка.  
2. Введите корпоративный логин и пароль.  
3. Дождитесь синхронизации клиента с сервером.  
4. После успешной авторизации откроется главное окно системы.

---

# 6. Интерфейс пользователя

Главное окно системы содержит:

1. **Главное меню**  
   - Сотрудники  
   - Расчёт зарплаты  
   - Премии  
   - Ведомости  
   - Отчёты  
   - Настройки  

2. **Рабочая область**, где отображаются формы редактирования и расчётов.  

3. **Статусная строка**, где отображается текущий пользователь и состояние подключения к серверу.

---

# 7. Работа с системой

## 7.1. Управление данными сотрудников

### 7.1.1. Просмотр списка сотрудников
1. Выберите пункт меню **Сотрудники → Список сотрудников**.  
2. На экране появится таблица со всеми сотрудниками.  
3. Доступна фильтрация по подразделению, должности и статусу.

### 7.1.2. Добавление нового сотрудника
1. Откройте **Сотрудники → Добавить сотрудника**.  
2. Заполните форму:  
   - ФИО  
   - Должность  
   - Подразделение  
   - Оклад  
   - Тип занятости  
   - Налоговые параметры  
3. Нажмите кнопку **Сохранить**.

### 7.1.3. Редактирование данных сотрудника
1. Выберите сотрудника из списка.  
2. Нажмите **Изменить**.  
3. Внесите корректировки.  
4. Нажмите **Сохранить**.

### 7.1.4. Удаление сотрудника
1. Откройте карточку сотрудника.  
2. Нажмите **Удалить**.  
3. Подтвердите действие.

*(Удаление доступно только пользователям с соответствующими правами.)*

---

## 7.2. Расчёт заработной платы

### 7.2.1. Единичный расчёт
1. Перейдите в раздел **Расчёт зарплаты**.  
2. Выберите сотрудника из выпадающего списка.  
3. Укажите расчётный период.  
4. Нажмите **Пересчитать**.

Система автоматически учитывает:
- оклад;  
- налоги;  
- страховые взносы;  
- удержания;  
- премии, назначенные сотруднику.

### 7.2.2. Массовый расчёт
1. Откройте **Ведомости → Массовый расчёт**.  
2. Укажите подразделение или весь штат.  
3. Выберите период.  
4. Нажмите **Рассчитать всех**.

Результат выводится в виде таблицы.

---

## 7.3. Работа с премиями

### 7.3.1. Назначение премии сотруднику
1. Откройте карточку сотрудника.  
2. Перейдите на вкладку **Премии**.  
3. Нажмите **Добавить премию**.  
4. Выберите тип премии:  
   - фиксированная;  
   - процент от оклада;  
   - коэффициентная.  
5. Укажите параметры.  
6. Сохраните изменения.

### 7.3.2. Отмена или изменение премии
Премию можно изменить или удалить, открыв её в карточке сотрудника.

---

# 8. Формирование отчётности

## 8.1. Формирование расчётной ведомости
1. Откройте меню **Ведомости**.  
2. Выберите период.  
3. Нажмите **Сформировать ведомость**.  
4. Для сохранения нажмите **Экспорт → PDF/XLSX**.

## 8.2. Экспорт данных
Доступные форматы:
- PDF  
- XLSX  
- CSV

# 9. Сообщения об ошибках

| Код | Описание | Действия пользователя |
|-----|----------|------------------------|
| E001 | Нет соединения с сервером | Проверить сеть, обратиться к администратору |
| E002 | Недостаточно прав | Обратиться к администратору за повышением роли |
| E003 | Некорректно заполнены поля | Проверить обязательные поля формы |
| E004 | Ошибка расчёта | Повторить попытку; при повторении — сообщить администратору |

# 10. Требования безопасности

- Не передавать свою учетную запись третьим лицам.  
- Не изменять критически важные данные без согласования.  
- Проводить выход из системы после завершения работы.  
- Использовать систему только в корпоративной сети.

# 11. Приложения

## Приложение А. Пример расчёта зарплаты сотрудника
**Исходные данные:**
- Оклад: 60 000 ₽  
- Премия: 10%  
- НДФЛ: 13%  

**Расчёт:**
- Премия = 60 000 × 0.10 = 6 000 ₽  
- Налоговая база = 66 000 ₽  
- НДФЛ = 8 580 ₽  
- Итог к выплате = 57 420 ₽
